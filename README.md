# El Laberinto

> A web-based narrative escape room experience built with HTML, CSS and JavaScript.
> 

## 📖 Description

*El Laberinto* is a small atmospheric web experience inspired by classic fantasy worlds.

The user progresses through a series of symbolic “rooms,” each containing a short narrative moment and a simple challenges that must be solved to advance.

The project focuses on immersive visuals, poetic text, and a clean, modular front-end architecture.

This repository contains the front‑end implementation of the experience, built with semantic HTML and modular CSS.

All design decisions, technical notes, and detailed documentation are available in the accompanying Notion workspace.

## 🚀 Getting Started

### Prerequisites

- A modern web browser

### Installation

```bash
  git clone https://github.com/danielPalmera/f5_g5_scape_room
```

# Open index.html in your browser

## Features

- Immersive landing page with hero image and overlay narrative card
- Multi‑room navigation with small interactive challenges
- Modular, scalable CSS architecture
- Responsive layout
- Consistent visual style based on a custom color palette and typography set
